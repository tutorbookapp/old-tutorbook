    @tutorbook/dialogs. EditLocationDialog

    Class that represents the "Edit Location" dialog/view that enables tutoring supervisors to:

    • Edit when their location is open
    • Configure service hour rounding for past appointments clocked at their location
    • Change their location's description (name cannot be changed... yet)
    • Add and remove supervisors for their location
